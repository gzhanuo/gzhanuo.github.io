Hexo + Github Pages个人博客搭建

3月 31 2019 理论&教程 24 分钟 读完 (约 3628 字)

一直有记录的习惯，现有资料收集或者是个人见解的表达。最初的时候会放在印象笔记，有道云笔记一类的软件中，手头仅有手机的时候书写就显得很不方便。有时候写了存在电脑硬盘，但不利于查阅，时间一久也不记得自己写过什么东西，中途电脑故障，换了台设备，剩下的就寥寥无几了。

市面上的博客很多，CSDN，博客园，简书等等，可以直接在线编辑发布，文章也可以被百度抓取。但往往不太自由，会受到平台的各种限制，过多的广告也会影响体验。而自己购买域名和服务器，搭建博客的成本太高。定期维护需要大量时间和精力，到后面很可能就不了了之。
